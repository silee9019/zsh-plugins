# zsh-plugins

[한국어 (Korean)](docs/README_ko.md)

A collection of custom Zsh plugins.

## Tech Stack

- Zsh / Shell

## Getting Started

Clone the repository and source the desired plugin in your `.zshrc`:

```zsh
source /path/to/zsh-plugins/<plugin-name>/<plugin-name>.plugin.zsh
```

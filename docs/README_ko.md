# zsh-plugins

[English (영어)](../README.md)

커스텀 Zsh 플러그인 모음.

## 기술 스택

- Zsh / Shell

## 시작하기

저장소를 클론하고 원하는 플러그인을 `.zshrc`에서 source합니다:

```zsh
source /path/to/zsh-plugins/<plugin-name>/<plugin-name>.plugin.zsh
```

## 플러그인

| 플러그인 | 설명 |
|----------|------|
